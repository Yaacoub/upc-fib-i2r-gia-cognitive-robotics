# Cognitive Robotics

This repository contains the experimental setup used for the I2R-GIA project.

## Pipeline A

**Configuration**:
Create a `.env` file inside the `Soar` folder and set your Gemini API key there.
```
GEMINI_API_KEY=your_api_key_here
```

**Execution**:
```bash
python3 Soar/interface.py
```
